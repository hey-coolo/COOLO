## // MISSION_PROTOCOL

**High-End Strategy + Execution.**

[coolo.co.nz](https://coolo.co.nz) is a digital experience designed to strip away the noise. It features a minimalist, terminal-inspired interface with interactive cursor sketching capabilities, allowing users to engage directly with the viewport canvas.

## // SYSTEM_ARCHITECTURE

* **Core:** [Next.js](https://nextjs.org/) (React Framework)
* **Styling:** Tailwind CSS
* **Interaction:** Custom Canvas / Drawing API
* **Motion:** Framer Motion

## // INITIALIZATION

To boot the system locally:

```bash
# 1. Clone the repository
git clone [https://github.com/your-username/coolo.git](https://github.com/your-username/coolo.git)

# 2. Access directory
cd coolo

# 3. Install dependencies
pnpm install

# 4. Execute launch sequence
pnpm dev
