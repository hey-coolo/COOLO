
# Images Directory

Upload your project images here.

## How to use:
1. Drag and drop your `.jpg`, `.png`, or `.webp` files into this folder.
2. In `src/constants.ts`, import them like this:

```typescript
import heroImage from './assets/images/hero.jpg';

// ... inside ASSETS object
hero: {
  viz: heroImage,
  // ...
}
```
